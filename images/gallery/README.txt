History gallery photos
=======================

The History page (history.html) loads its photos from THIS folder.

Two ways to fill it:

EASIEST — run the downloader
  From the main vcsar1 folder, run  download-history-images.command  once.
  It pulls every History/gallery photo from your current vcsar1.org into here.
  (Mac: if double-clicking is blocked, right-click the file ▸ Open ▸ Open,
   or open Terminal and run:  bash download-history-images.command )

OR — drop in your own copies
  If you already have these photos, copy them in using the SAME filenames the
  page expects. The grid uses a "-634x476" thumbnail plus a full-size version:
    SARwebsite_History-Gallery-1-634x476.jpg   (thumbnail shown in the grid)
    SARwebsite_History-Gallery-1.jpg           (full size, opens in lightbox)
    ...through 27...
  plus DSC_7775 / DSC_7708 / DSC_7739 / DSC_7763 / DSC_7753 (-634x476 + -scaled),
  history-bg01/02, Scan10012_flooding, Scan10052_onmountain, Scan10042_firsttruck,
  history_onmountain2, history-oldSARbadge-800x800-1, history-Scan10047_crew,
  SARwebsite_History-jeep-1, and SARwebsite_History-hd01-1-2 (the page banner).

Until this folder is filled, the History page's photo spots appear blank.
Once filled, the site no longer depends on WordPress for any image.
